## 레포 구분 (중요)

- **코드** → `Oven-up` 레포
- **문서** → `Oven-up-repo` 레포 (이 문서들이 갈 곳)
- **PR 규칙** → `QT-AI-2nd-Team-Project`의 `.github` 참고
- **문서 형식** → `2nd-Team-Project` 구조만 참고

## 작업 원칙
- 작업 전 항상 `git pull`로 최신화 후 시작.
- 한 번에 하나씩, 가장 작은 것부터 완성.
