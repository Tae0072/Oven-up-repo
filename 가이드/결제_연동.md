# 결제 연동 가이드 (PortOne)

> 오븐업 결제는 **개발용 모의(mock) 결제**가 기본이에요. 실제 카드·간편결제를 붙이려면
> 결제대행사 **PortOne(포트원, 구 아임포트)** 계정과 키가 필요합니다. 이 문서대로 하면 돼요.
> 버전: v0.1 · 작성일: 2026-07-04

---

## 0. 지금 상태 (중요)

- 서버 설정 `app.payment.mock=true` (기본) → **실제 결제 없이** 주문을 '결제완료'로 처리해요. 개발·데모용.
- 실제 결제를 켜려면 `app.payment.mock=false` 로 바꾸고 아래 준비물을 설정합니다.
- ⚠️ **실제 결제는 진짜 돈이 오가요.** 반드시 PortOne "테스트 모드"로 먼저 충분히 확인한 뒤 운영에 켜세요.

---

## 1. 결제 흐름 한눈에

```
[앱] 주문서 '결제하기'
      │  1) POST /api/orders            → 주문 생성(상태: 결제대기), orderId·금액 받음
      ▼
[앱] 결제 화면(S7)에서 결제수단 선택
      │  2) (실연동) PortOne 결제창 호출 → 결제 → paymentId 받음
      │  3) POST /api/orders/{id}/pay { method, paymentRef=paymentId }
      ▼
[서버] PortOne에 조회해서 "결제완료·금액일치" 확인 → 주문 상태 '결제완료'
      ▼
[앱] 주문완료 화면(S8)
```

- **핵심 안전장치**: 결제 금액은 화면 값이 아니라 **서버가 계산한 주문 금액**과 대조해요(조작 방지). 서버가 PortOne에 실제 결제금액을 물어보고 다르면 거부합니다.

---

## 2. 준비물 (PortOne 콘솔)

1. [PortOne](https://portone.io) 가입 → 콘솔 로그인.
2. **결제연동(PG) 설정**: 사용할 결제수단(카드/카카오페이/네이버페이/토스페이/삼성페이) 채널 추가.
   - 각 간편결제사는 별도 심사/계약이 필요할 수 있어요. **시작은 '카드 + 카카오페이(테스트)'** 정도를 권장.
3. **V2 API Secret** 발급: 콘솔 → 결제 연동 → API Keys. (서버가 결제를 검증할 때 사용)
4. 앱(프론트)용 **스토어 ID / 채널 키**: 결제창을 띄울 때 필요.

---

## 3. 서버 설정 (실연동 켜기)

`server/src/main/resources/application.properties` 는 그대로 두고, **환경변수**로 켭니다(키를 코드에 넣지 않기).

- `app.payment.mock=false`
- `PORTONE_API_SECRET=<PortOne V2 API Secret>`

실행 예:
```bat
set PORTONE_API_SECRET=여기에_시크릿
gradlew.bat bootRun --args="--app.payment.mock=false"
```

서버는 `RealPaymentVerifier`가 `https://api.portone.io/payments/{paymentId}` 를 조회해
`status=PAID` 이고 `amount.total` 이 주문 금액과 같은지 확인합니다.

> 이미 만들어 둔 파일: `server/.../payment/RealPaymentVerifier.java`
> PortOne 응답 형식이 다르면 이 파일의 status/amount 파싱 부분만 맞춰 주면 돼요.

---

## 4. 앱(프론트) 결제창 붙이기 (요약)

지금 앱의 결제 화면은 **모의 결제**라 바로 서버로 `pay`를 호출해요. 실연동에서는 그 사이에 결제창을 넣습니다.

1. Flutter에 PortOne 결제 SDK(웹은 PortOne JS, 앱은 `portone_flutter` 등) 추가.
2. 결제 화면에서 결제수단 선택 후 **PortOne 결제창 호출** → 성공 시 `paymentId` 획득.
3. 그 `paymentId` 를 `OrderApi.payOrder(paymentRef: paymentId)` 로 전달.
4. 서버가 검증 후 '결제완료' → 주문완료 화면.

> 코드 위치: `app/lib/screens/payment_page.dart` 의 `_pay()` — "// 실제 연동 시엔 여기서 PortOne 결제창을 띄우고 paymentRef를 받아 전달" 주석 지점.
