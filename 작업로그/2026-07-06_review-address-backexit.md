- 서버에는 이미 구매이력 검증이 있었음(결제완료 주문에 해당 메뉴 포함 여부, NOT_PURCHASED)

## 2. 홈·메뉴 좌측 상단 주소 선택 (배민식)
- state/address_store.dart: 현재 주소 전역 상태 (로그인 시 서버 주소 로드, 변경 시 서버 저장)
- widgets/address_title.dart: 위치 아이콘 + 주소(말줄임) + ▾, 탭하면 다음 주소검색창 → 즉시 변경
- 홈·메뉴 AppBar 타이틀에 적용, 주소 미설정 시 "주소 설정" 표시, 로그아웃 시 초기화

## 3. 홈에서 뒤로가기 두 번 → 앱 종료 (안드로이드)
- MainShell PopScope: 홈이 아닌 탭에서 뒤로가기 → 홈 탭으로 이동
- 홈에서 뒤로가기 → "한 번 더 누르면 앱이 종료돼요" 스낵바 → 2초 내 재입력 시 종료(SystemNavigator.pop)
